# 💰 Expense Tracker

A modern **Expense Tracker Web Application** built to help users manage daily income and expenses efficiently.  
This project allows users to **add transactions, categorize spending, track balance, and monitor financial activity** through a clean and simple interface.

The goal of this project is to provide a lightweight and user-friendly personal finance management system for tracking expenses and understanding spending habits.

---

## 🚀 Features

- ➕ Add income and expense transactions
- 🗑️ Delete transactions
- 📋 View complete transaction history
- 💵 Track total balance
- 📈 Separate income and expense summaries
- 🏷️ Categorize transactions
- 📅 Manage daily/monthly expenses
- 💾 Data persistence for saved transactions 
- 📱 Responsive user interface for desktop and mobile
- ⚡ Simple, fast, and easy-to-use expense management system

---

## 📌 Project Overview

**Expense Tracker** is a personal finance web application where users can record their financial transactions and keep track of their money flow.

## 🎯 Core Functionalities

### 1. Add Transactions
Users can add:
- **Income**
- **Expenses**

Each transaction can include:
- Title / description
- Amount
- Type (income or expense)


### 2. View Transaction History
All added transactions are displayed in a list so users can review their financial activity.

### 3. Balance Calculation
The app automatically calculates:
- **Current Balance**
- **Total Income**
- **Total Expenses**

### 4. Delete Transactions
Users can remove transactions when needed, and the totals update automatically.

### 5. Expense Management
The project helps users understand how much they are spending and where their money is going.
